A New Way
Required Tool: Text Editor and Google Chrome Web Inspector
In this exercise, you will learn how to represent an ordinary HTML tag that you learned in Lecture 1 using just divs, spans, and a pinch of CSS.
Assignment

Technically, almost all HTML tags can be represented just using one tag: <div>To understand this, you must first know an important concept in HTML: inline and block elements. Simply put, inline elements stack horizontally and block elements stack vertically. An example of inline element is <span>. There are a lot of block elements, such as <p>, <img>, and <div>.

A powerful concept behind inline and block elements is that almost all HTML tags are essentially the same, and can be styled the way you want just by using CSS. For example, <p> tag is the same as <div> tag just with a few additional CSS properties. This is because <div> tag is a pure HTML block element that stacks vertically without any CSS property attached to it.
